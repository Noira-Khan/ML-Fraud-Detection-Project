# Fraud Detection Pipeline

A Python project scaffold for building a fraud detection data pipeline.

## Structure

- `src/fraud_detection_pipeline/`: package source code
- `tests/`: unit tests
- `.github/workflows/`: CI workflow

## Setup

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
pip install -e .
```

## Testing

```powershell
pytest
```
