## Overview

An end-to-end Machine Learning pipeline for detecting fraudulent credit card transactions.

The project covers:

- Data Loading
- Exploratory Data Analysis
- Data Preprocessing
- Feature Engineering
- Class Imbalance Handling
- Model Training
- Hyperparameter Tuning
- Streamlit User Interface
- Fraud Prediction
